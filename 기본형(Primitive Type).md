## 기본형 타입 - 정수형, 문자형, 실수형, 논리형


정수형|long|int|short|byte
---|---|---|---|---
범위|-922경 ~ 922경|-21억 ~ 21억|-32,768 ~ 32,767|-128~127
메모리 크기|8byte|4byte|2byte|1byte

> long 변수가 int형 범위를 넘을 시, 뒤에 L을 붙여줘야함. 그렇지 않으면 컴파일 오류 발생
>
> ex) long a = 2134294329423423L;

---
  
문자형|char|
---|---
범위|0 ~ 65535
메모리 크기|2byte

```java
public class lab {
    public static void main(String[] args) {

        // 문자, 10진수, 16진수로 저장
        
        char ch1 = 'A'; // 문자 'A' 직접 저장

        char ch2 = 65; // 10진수로 'A' 저장, 범위: 0 ~ 65535

        char ch3 = '\u0041'; // 16진수로 'A' 저장


        // char형을 int형으로 변환시, 10진수에 해당하는 유니코드가 나옴.
        
        int n1 = ch1; // 65
        int n2 = ch2; // 65
        int n3 = ch3; // 65
        
    }
}

```

> 10진수 65 ~ 90은 문자 A부터 Z까지이고,
>
> 10진수 97 ~ 122는 a부터 z까지다.

실수형|float|double
---|---|---
범위|1.4 * 10E-45 ~ 3.4 * 10E38|4.9 * 10E-324 ~ 1.8 * 10E308
유효숫자 자리수| 7자리 |15자리
메모리 크기|4byte|8byte
정확도|낮음|높음
> float은 사용할 때, 숫자 뒤에 f 또는 F를 붙여야 한다.
>
> 유효숫자 자리수를 넘어갈 때 정확도가 떨어진다.

---|---|---
범위|1byte(8bit)|1byte(8bit)
